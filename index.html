Fast Notes

Android 4.1+ приложение для быстрых заметок из мультимедийных блоков. Кроме текстовых элементов, возможно добавление фотографий с камеры и из галереи. Для записи звука используется встроенный диктофон с AAC кодированием.

PWA Template

Приложение-шаблон. Позволяет преобразовать сайт в Pure Web Application для Android 4.1+. Загружает меню в json формате с сайта и на его основе отображает боковой навигатор и панель вкладок. Поддерживает кэширование, а для быстрых сетей есть функция предзагрузки.

NetSpeakerRx

Легковесная консольная программа для передачи звука по локальной сети. Обработка несложным кодеком позволяет передавать звук почти без задержки и не нагружать процессор. Содержит клиент и сервер в едином пакете и поддерживает несколько подключений одновременно. Успешно используется по WiFi и восстанавливается из состояния сна.

NetworkingRx

Библиотека сетевых компонентов в реактивном стиле. Используется в NetSpeakerRx для взаимодействия по UDP протоколу. Несколько уровней абстракции позволяют легко использовать компоненты в других проектах, например "Умного дома". Возможна работа на Android.
