Ytdl-gui

Легковесное портабельное приложение для скачивания видео с хостингов. Является оберткой над консольной программой youtube-dl и совместима с её форками, например yt-dlp. Поддержка сайтов полностью зависит от выбранного downloader'а. Обычно поддерживаются Youtube, Yandex Video, Vk.com, Ok.ru и другие. Перед загрузкой GUI позволяет выбрать качество видео и его формат (mp4/webm).

Fast Notes

Приложение для быстрых заметок из мультимедийных блоков. Кроме текстовых элементов, возможно добавление фотографий с камеры и из галереи. Для записи качественного звука используется собственный диктофон с AAC кодированием. Есть возможность пакетного добавления и распознавания штрихкодов. Не требует доступа в интернет. Работает на устройствах с Android 4.1 и выше. Поддерживает запросы Runtime разрешений на Android 6.0+

PWA Template

Приложение-шаблон. Позволяет преобразовать сайт в Pure Web Application для устройств с версией Android 4.1+. Загружает меню в json формате с сайта или встроенных ресурсов и на его основе отображает боковой навигатор и панель вкладок. Поддерживает кэширование, а для быстрых сетей есть функция предзагрузки соседних страниц.

NetSpeakerRx

Легковесная консольная программа для передачи звука по локальной сети. Обработка несложным кодеком позволяет передавать звук почти без задержки и не нагружать процессор. Содержит клиент и сервер в едином пакете и поддерживает несколько подключений одновременно. Успешно используется по WiFi и восстанавливается из состояния сна.

NetworkingRx

Библиотека сетевых компонентов в реактивном стиле. Используется в NetSpeakerRx для взаимодействия по UDP протоколу. Несколько уровней абстракции позволяют легко использовать компоненты в других проектах, например "Умного дома". Возможна работа на Android.
